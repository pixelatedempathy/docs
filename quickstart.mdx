## ğŸ’» For Developers

> *Set up the full platform locally for development and contribution.*

<Info>

**Prerequisites:** Node.js 19+ â€¢ Python 3.11+ â€¢ pnpm â€¢ MongoDB â€¢ Redis â€¢
Docker (optional)

</Info>

<Steps>

<Step title="1. Clone the repository">

```bash
git clone https://github.com/pixelatedempathy/pixelated-empathy.git
cd pixelated-empathy
```

</Step>

<Step title="2. Install dependencies">

```bash
# Frontend
pnpm install

# Python AI/ML
cd ai-services
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
cd ..
```

</Step>

<Step title="3. Configure environment">

```bash
cp .env.example .env
```

Key variables:

```bash
# Database
MONGODB_URI=mongodb://localhost:27017/pixelated-empathy
REDIS_URL=redis://localhost:6379

# AI Services
TOGETHER_API_KEY=your_together_api_key
FINE_TUNED_THERAPEUTIC_MODEL=meta-llama-3-8b-instruct

# Auth
AUTH_SECRET=your_auth_secret_here
```

<Warning>

Never commit `.env` to version control!

</Warning>

</Step>

<Step title="4. Start development server">

```bash
# Full stack
pnpm dev

# Or with AI services (separate terminal)
cd ai-services
source .venv/bin/activate
python -m uvicorn main:app --reload --port 8000
```

ğŸŒ **Astro:** <http://localhost:4321>

</Step>

<Step title="5. Run tests">

```bash
# Unit & integration
pnpm test

# E2E
pnpm test:e2e

# Python AI
cd ai-services
python -m pytest
```

</Step>

</Steps>
