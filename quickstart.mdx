## 🧠 For Therapists & Counselors

> _Get from sign-up to your first AI training session in under five minutes._
<Steps>

<Step title="1. Create your account">

Navigate to **[pixelatedempathy.com](https://pixelatedempathy.com)** and click
**Get Started**.

## What you'll need

- 📧 Email or institutional SSO
- 🏷️ Professional role (therapist, counselor, psychologist, student)
- 📊 Years of clinical experience
- 🎯 Areas of specialization (optional)

<Tip>

If your organization has already set up Pixelated Empathy, ask your admin for an
**invitation link** — it auto-connects you to your institution's training
program.

</Tip>

</Step>

<Step title="2. Complete your clinical profile">

Calibrate training scenarios to your experience level:

1. **Select training focus** — Crisis intervention, trauma-informed care,
   motivational interviewing, couples therapy, substance abuse
1. **Set difficulty** — Beginner (guided) → Master (complex, multi-issue)
1. **Choose feedback** — Real-time, post-session, or both

<Info>

Update anytime via **Settings → Clinical Profile**. The platform adapts as you
grow.

</Info>

</Step>

<Step title="3. Launch your first session">

Click **Enter The Empathy Gym™** from your dashboard:

1. **Pick a scenario** — Anxiety, depression, trauma, crisis, personality
   disorders
1. **Review client profile** — Emotional baseline, personality traits, triggers,
   goals
1. **Start the conversation** — Text or voice. AI client adapts dynamically.

<Warning>

⚠️ Crisis scenarios include sensitive content with built-in safety protocols.
You can skip or adjust content preferences.

</Warning>

</Step>

<Step title="4. Review your feedback">

Get a comprehensive post-session report:

| Metric                      | Description                          |
| --------------------------- | ------------------------------------ |
| 🧡 **Empathy Score**        | Emotional attunement (0.0–1.0)       |
| 🔧 **Technique Analysis**   | CBT, DBT, MI, psychodynamic usage    |
| 🛡️ **Bias Check**           | Demographic bias pattern detection   |
| 🚨 **Crisis Response**      | Crisis signal identification speed   |
| 🤝 **Therapeutic Alliance** | Rapport strength (task, bond, goals) |
| 📈 **Skill Progression**    | Historical performance comparison    |

<Tip>

Click any metric to see specific examples from the conversation + evidence-based
improvement suggestions.

</Tip>

</Step>

</Steps>

## 👁️ For Supervisors & Institutions

> _Set up your organization's training environment and monitor trainee
> progress._
<Steps>

<Step title="1. Register your organization">

Contact
**[enterprise@pixelatedempathy.com](mailto:enterprise@pixelatedempathy.com)** or
visit the **[Enterprise Portal](https://pixelatedempathy.com/enterprise)**.

## Required

- 🏢 Organization name & type
- 👥 Estimated trainee count
- 📋 Compliance requirements (HIPAA, FERPA, IRB)
- 🔐 SSO/identity provider details (if applicable)

You'll receive admin credentials + access to the **Supervisor Dashboard**.

</Step>

<Step title="2. Configure your training program">

<AccordionGroup>

<Accordion title="Create trainee cohorts" icon="users">

Organize by class year, specialization, or rotation. Each cohort gets:

- Required scenario sequence
- Competency benchmarks
- Assessment frequency
- Difficulty levels

**Path:** Dashboard → Cohorts → Create New Cohort

</Accordion>

<Accordion title="Assign training pathways" icon="route">

## Built-in programs

- _Basic Therapeutic Skills_ — 5 progressive sessions for beginners
- _Crisis Intervention Specialist_ — 8 complex scenarios for experienced
  clinicians
- _Custom Pathway_ — Define your own sequence

**Path:** Dashboard → Training Programs → Assign to Cohort

</Accordion>

<Accordion title="Set competency thresholds" icon="chart-bar">

Minimum levels across 8 core skills:

| Skill                     | Description                | Levels |
| ------------------------- | -------------------------- | ------ |
| Rapport Building          | Connection & alliance      | 1-5    |
| Active Listening          | Understanding & reflection | 1-5    |
| Empathy Demonstration     | Emotional attunement       | 1-5    |
| Crisis Management         | Safety assessment          | 1-5    |
| Resistance Handling       | Therapeutic resistance     | 1-5    |
| Boundary Setting          | Professional limits        | 1-5    |
| Therapeutic Confrontation | Appropriate challenging    | 1-5    |
| Cultural Competence       | Diverse backgrounds        | 1-5    |

</Accordion>

<Accordion title="Invite trainees" icon="envelope">

1. Dashboard → Cohorts → [Cohort Name] → Members
1. Click **Invite Members**
1. Enter emails or upload CSV
1. Select role & training pathway
1. Send invitations

Trainees auto-enroll upon registration.

</Accordion>

</AccordionGroup>

</Step>

<Step title="3. Monitor trainee progress">

Real-time visibility into performance:

- 👁️ **Live Observation** — Watch sessions with competency overlay
- 🚨 **Intervention Alerts** — Notifications for skill deficits
- 📊 **Progress Reports** — Analytics, patterns, breakthroughs
- 📋 **Development Plans** — Personalized growth recommendations

<Info>

All data is encrypted & access-controlled. See
[Compliance & Security](/compliance/security).

</Info>

</Step>

</Steps>

## 💻 For Developers

> _Set up the full platform locally for development and contribution._
<Info>

**Prerequisites:** Node.js 19+ • Python 3.11+ • pnpm • MongoDB • Redis • Docker
(optional)

</Info>

<Steps>

<Step title="1. Clone the repository">

```bash text
git clone <https://github.com/pixelatedempathy/pixelated-empathy.git>
cd pixelated-empathy

```

</Step>

<Step title="2. Install dependencies">

```bash

# Frontend
pnpm install

## Python AI/ML
cd ai-services
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
cd ..

```

</Step>

<Step title="3. Configure environment">

```bash text

cp .env.example .env

``` text


Key variables:


```bash

## Database
MONGODB_URI=mongodb://localhost:27017/pixelated-empathy
REDIS_URL=redis://localhost:6379

## AI Services
TOGETHER_API_KEY=your_together_api_key
FINE_TUNED_THERAPEUTIC_MODEL=meta-llama-3-8b-instruct

## Auth
AUTH_SECRET=your_auth_secret_here

```

<Warning>

Never commit `.env` to version control!

</Warning>

</Step>

<Step title="4. Start development server">

```bash

## Full stack
pnpm dev

## Or with AI services (separate terminal)
cd ai-services
source .venv/bin/activate
python -m uvicorn main:app --reload --port 8000

```

🌐 **Astro:** <http://localhost:4321>

</Step>

<Step title="5. Run tests">

```bash

## Unit & integration
pnpm test

## E2E
pnpm test:e2e

## Python AI
cd ai-services
python -m pytest

```

</Step>

</Steps>

### 🔧 Developer Quick Reference

<AccordionGroup>

<Accordion title="Project structure" icon="folder-tree">

```text

pixelated-empathy/
├── src/                    # Astro + React frontend
│   ├── components/          # Dashboard, simulator, etc.
│   ├── pages/              # Astro pages & API routes
│   ├── lib/                # AI services, auth, API clients
│   ├── simulator/           # Empathy Gym™ engine
│   └── hooks/              # React hooks
├── ai-services/            # Python AI/ML microservices
├── ai/                     # Training pipelines & models
├── docs/                  # This documentation (Astro Starlight)
└── tests/                 # Test suites
