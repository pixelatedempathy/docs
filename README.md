## Key Concepts

### Emotional Intelligence Framework

We use two scientifically-validated frameworks:

1. **Plutchik's Wheel of Emotions** â€” 8 primary emotions with intensity
   variations
2. **Big Five (OCEAN)** â€” Openness, Conscientiousness, Extraversion,
   Agreeableness, Neuroticism

All emotion scores are normalized to `0.0â€“1.0` floats for consistent processing.

### The EARS Framework

Every AI response is evaluated against:

- **E**mpathy â€” Emotional attunement score
- **A**uthenticity â€” Genuine vs. scripted response ratio
- **R**elevance â€” Contextual appropriateness
- **S**afety â€” Crisis detection & psychological safety

### Bias Detection

Real-time monitoring for 40+ bias categories:

- Demographic (age, gender, ethnicity)
- Linguistic (dialect, accent, vocabulary)
- Cultural (norms, values, communication styles)
- Socioeconomic (education, income, occupation)

### Prerequisites

- Node.js 19+
- pnpm 8+

```bash

npm i -g mintlify

# Run docs locally
cd docs
mintlify dev
```

Documentation will be available at `http://localhost:3333`

### Build for Production

```bash

```
